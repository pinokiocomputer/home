What is a verified script?

1. Verified Authors: Verified scripts are scripts from verified publishers.
2. How to get verified: Start by publishing as a community script and ask to be featured on Pinokio community Discord

What is a community script?

1. No filter: This is a raw feed of pinokio scripts published by the community.
2. Automated: All github repositories tagged as 'pinokio' show up here automatically.
3. May not work: Remember, these scripts have NOT been vetted, and they may not work as you expect. If you decide to try these scripts, make sure to find the publisher on Pinokio community Discord and ask first.

Should you install?

1. Why you need to be careful: Since Pinokio scripts can run system commands, you need to be careful with what you install and run.
2. [WARNING!!!] DO NOT install a community script unless you TRUST the person who published it.
3. Ask: Even if you trust the publisher, ask around in the Pinokio community Discord before installing anything. Ideally, the publisher should be on Discord.
4. Verify yourself: If you're adventureous, you can read through the script source code to verify yourself there's nothing wrong, and then install.

How to get listed & verified

1. Listing: To list your script here, simply add a topic named pinokio on your Github repo. It will automatically show up here. Learn more
2. When verification? If the publisher makes a request, or enough people request a community script to be verified, it may be promoted to the "Verified" status through Pinokio Factory
3. How verification works: The original publisher doesn't need to do anything (except for publishing as a community script). The Pinokio Factory Github organization may fork the original code, freeze the code, and THEN feature the frozen script on the "Verified" section.
